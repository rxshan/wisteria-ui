# ConfigProvider

```tsx
import { ConfigProvider } from 'wisteria-ui';
```

## 用法

### 主题切换

```tsx
<ConfigProvider theme="light" />
```

<Callout type="info">组件库内置 **亮色** 和 **暗色** 主题。</Callout>

<CodeDemoArea label="Theme">
  <SwitchThemeDemo />
</CodeDemoArea>

### 动态主题

<Callout type="info">动态主题切换基于 **CSS 变量** 实现。</Callout>

```tsx
<ConfigProvider
  theme="light"
  cssVars={{
    primary: '#6750a4',
    primaryContainer: '#e9ddff',
    secondary: '#625b71',
    secondaryContainer: '#e8def8',
    tertiary: '#7e5260',
    tertoaryContainer: '#ffd9e3'
  }}
/>
```

<CodeDemoArea label="Dynamic Color">

</CodeDemoArea>
