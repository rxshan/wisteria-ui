
## 主题切换

```tsx
<ConfigProvider theme="light" />
```

## 动态主题

> 通过 CSS 变量实现

```tsx
<ConfigProvider
  theme="light"
  cssVars={{
    primary: '#6750a4',
    primaryContainer: '#e9ddff',
    secondary: '#625b71',
    secondaryContainer: '#e8def8',
    tertiary: '#7e5260',
    tertoaryContainer: '#ffd9e3'
  }}
/>
```
