# Provider

```tsx
import { WisteriaProvider } from 'wisteria-ui';
```

## 用法

### 主题切换

```tsx
<WisteriaProvider theme={{ scheme: 'light' }} />
```

<Callout type="info">组件库内置 **亮色** 和 **暗色** 主题。</Callout>

### 获取当前主题

```tsx
import { useTheme } from 'wisteria-ui';

const App = () => {
  const theme = useTheme();

  return <main>App</main>;
};
```

### 动态主题

<Callout type="info">动态主题切换基于 **CSS 变量** 实现。</Callout>

```tsx
<WisteriaProvider
  theme={{
    scheme: 'light',
    colors: {
      primary: '#6750a4',
      primaryContainer: '#e9ddff',
      secondary: '#625b71',
      secondaryContainer: '#e8def8',
      tertiary: '#7e5260',
      tertoaryContainer: '#ffd9e3'
    }
  }}
/>
```

<Callout>
  当前代码仅为示例，更多样式设置请查看 **ThemeColors** 类型，具体含义可在
  [Material Design](https://m3.material.io/styles/color/the-color-system/tokens)
  文档中查看
</Callout>
