# CardActions

```tsx
import { Card, CardActions } from 'wisteria-ui';
```

## 用法

```tsx
<Card style={{ width: '50%' }}>
  <Card.Title title="Card Actions" />
  <Card.Actions placement="right">
    <Button variant="text">SHARE</Button>
    <Button variant="text">LIKE</Button>
  </Card.Actions>
</Card>
```
