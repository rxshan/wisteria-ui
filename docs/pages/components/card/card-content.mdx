# CardContent

```tsx
import { Card, CardContent } from 'wisteria-ui';
```

## 用法

```tsx
<Card style={{ width: '50%' }}>
  <Card.Title title="Card Content" />
  <Card.Content>
    Lorem ipsum, dolor sit amet consectetur adipisicing elit. Animi fugiat
    facere sapiente error hic, id nobis repellendus natus atque, nostrum dolorem
    laboriosam ex cumque deleniti expedita quo suscipit provident et?
  </Card.Content>
</Card>
```
