## Variant

<Callout type="info">
  Button 共有 5
  种样式类型，分别为：**Elevated**、**Filled**、**Tonal**、**Outlined**、**Text**。
</Callout>

```tsx filename="tsx"
<Button>Elevated</Button>
<Button variant="filled">Filled</Button>
<Button variant="tonal">Tonal</Button>
<Button variant="outlined">Outlined</Button>
<Button variant="text">Text</Button>
```

## API

<Callout type="info">**Button** 组件支持 **html button** 所有属性。</Callout>

<ConfigAPI
  list={[
    {
      property: 'block',
      description: '是否撑满容器',
      typedef: 'boolean'
    },
    {
      property: 'color',
      description: '自定义颜色',
      typedef: ' string'
    },
    {
      property: 'corner',
      description: '设置圆角',
      typedef: 'square | rounded | circular',
      defaultValue: 'rounded'
    },
    {
      property: 'variant',
      description: '按钮类型',
      typedef: 'elevated | filled | tonal | outlined | text',
      defaultValue: 'elevated'
    }
  ]}
/>
